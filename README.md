# Lead-Conversion-Prediction
## 📌 Project Overview
This project aims to predict whether a lead will convert into a customer using machine learning techniques.

## 📊 Dataset
The dataset contains historical marketing and sales data including lead source, stage, engagement level, etc.

## ⚙️ Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Model Building

## 🤖 Model Used
- Random Forest Classifier

## 📈 Results
- Achieved good accuracy in predicting lead conversion.
- Helps businesses identify high-potential leads.

## 🚀 Conclusion
This model can help companies improve marketing strategies and increase conversion rates.
